Paragraphs

A paragraph (from the Greek paragraphos, “to write beside” or “written beside”) is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text. It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom. Said no one, ever.

Additional questions

What is selector's specificity and how is it calculated?Selector specificity is a weight that determines which styles are applied to an element when multiple conflicting styles exist. It is calculated based on the combination of type selectors, class selectors, and ID selectors present in the selector. The specificity is represented as a four-part number, with each part indicating the specificity of a different type of selector.

What are complex selectors? Complex selectors are CSS selectors that combine multiple simple selectors to target elements with specific characteristics or relationships in the document tree. Examples include descendant selectors, child selectors, sibling selectors, attribute selectors, and pseudo-classes.

What is the difference between pseudo-classes and pseudo-elements? Pseudo-classes are keywords added to selectors that specify a special state of the selected elements. Examples include :hover, :active, :focus, and :first-child. Pseudo-elements, on the other hand, are keywords added to selectors that style a specific part of an element's content. Examples include ::before, ::after, ::first-letter, and ::selection.
